# Домашнє завдання 10 — Лінійне програмування та рандомізовані алгоритми

## Завдання 1: Оптимізація виробництва
Компанія виробляє два напої: **Лимонад** (Вода, Цукор, Лимонний сік) та **Фруктовий сік** (Фруктове пюре, Вода).

**Обмеження ресурсів:**
- Вода: 100
- Цукор: 50
- Лимонний сік: 30
- Фруктове пюре: 40

**Витрати на одиницю продукції:**
- Лимонад: 2 Води, 1 Цукру, 1 Лимонного соку
- Фруктовий сік: 1 Води, 2 Фруктового пюре

**Мета:** максимізувати загальну кількість продуктів.  

**Результат:** оптимальна кількість одиниць Лимонаду та Фруктового соку і максимальна кількість продуктів.

## Завдання 2: Інтеграл методом Монте-Карло
Обчислити інтеграл \( f(x) = x^2 \) на [0, 2] методом Монте-Карло.

**Підхід:**
1. Генерація N випадкових точок у прямокутнику.
2. Підрахунок точок під кривою.
3. Обчислення площі як частки точок під кривою від загальної кількості, помножене на площу прямокутника.
4. Повторення експерименту для підвищення точності.

**Результат:**  
- Середня площа методом Монте-Карло: ~2.667  
- Аналітичний результат: 2.6667  

**Висновок:** метод Монте-Карло дає близьке наближення до точного значення інтеграла.
